# asdf-pivotal
ASDF plugin to install tools from github.

## Supported Tools
- bosh
- credhub
- fly
- om
- pivnet
- bbr & bbr-s3-config-validator

## Installation
To manually install use:
```
asdf plugin-add <tool name> https://github.com/laidbackware/asdf-pivotal.git
```
Soon to hopefully be built into ASDF.
